Game & Simulation Development Manual

The Suprannua Engine plainly uses regular 2D polygons and blocks to represent game entities or agents of physical or algorithmic simulations. It doesn't use a development suite, nor does it offer a flexible means of integrating newer core componentry to make it into anything but a superannuated 2D canvas.

You don't need to edit the Engine files to write a simple game or simulation using the Suprannua Engine, but you can if you have modifications in mind. Neither do you need to know the Engine inside out.

What you do need to know is;

  • Very basic C/C++
  • Suprannua's game objects.
  • Suprannua's functions.

The Suprannua Engine acts as a framework where you can edit game object data, as rigidly defined by the Engine, using almost nothing but functions. Depending on the game, most of the functions can be called in a simple sequence, but knowledge of C (at the very least) is needed for situations where you might want to loop functions (e.g. causing multiple polygons to fall under gravity) and test for logical conditions (checking whether the player has past a checkpoint).

Game Objects

The game objects are Polygons, Blocks, Texts and the Camera. All game objects are instances of type defined structs with other structs and variables inside. They all contain the Vertex struct to represent x and y positions against a world map. The world map and rectangular objects like the Camera and Blocks all contain the Rect struct to represent width and height.

Game Object Structs

Polygons and Blocks are the only objects that contain the Properties struct, which describes the object's;

  • Classification (Whether it is a background, entity, etc)
  • Colour (Including alpha)
  • Edges (Geometric sides)
  • Angle (Only really applicable to Polygons)
  • BouncePercentage (Elasticity)
  • Mass
  • xVelocity
  • yVelocity

Property Struct

Text is different where in addition to position, it stores a string with colour and either a entity or heads up display classification (to pin to the camera or world).
